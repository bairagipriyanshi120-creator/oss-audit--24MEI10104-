## Introduction
This project demonstrates basic Linux commands and shell scripting concepts including system information retrieval, package checking, user interaction, file handling, and arithmetic operations.

## Scripts Description
1. **script1.sh** — System Identity Report (shows kernel, user, uptime)
2. **script2.sh** — FOSS Package Inspector (checks if git is installed)
3. **script3.sh** — User Input Script (takes input from user)
4. **script4.sh** — File Checker (checks if file exists)
5. **script5.sh** — Simple Calculator (performs arithmetic operations)
   
## How to Run Scripts on Linux
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh

./script1.sh
./script2.sh
./script3.sh
./script4.sh
./script5.sh
